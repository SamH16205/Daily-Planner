# Daily Planner

## Author: 
Sam Halloum

## Application purpose:
The application allows users to organize their workday by the hour and keep themselves on track with their goals by displaying the current hour and what they planned for the day.

## Application use: 
Each hour between 9AM and 5PM are given a textbox and a button. Users can enter what they want to accomplish for each hour and click the save button to save the text. Additionally, the textboxes are displayed as grey if the hour has already passed, red if it is the current hour, and green if they are in the future.
